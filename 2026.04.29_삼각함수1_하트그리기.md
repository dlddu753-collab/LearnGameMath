# [2026.04.29] 삼각함수로 하트 그리기

## 핵심 개념
- 삼각함수를 응용한 하트 방정식을 사용해 하트 형상을 출력
> $$하트 방정식$$
> $$x = 16\sin^3(t)$$
> $$y = 13\cos(t) - 5\cos(2t) - 2\cos(3t) - \cos(4t)$$

## 수학적 개념
> 빗변 c, 밑변 a, 높이, b 라 하고,  단위원에서 x축과 빗변(원의 반지름)이 이루는 각을 $\theta$ 라고 할때
- 삼각함수 : 직각 삼각형을 데카르트 좌표계 상에 배치하고 사잇각 범위를 실수 전체로 확장한 대응관계
> $$sin = \frac{b}{c} , cos = a / c , tan = b / a$$
- 단위원 : 반지름이 1인 평면 위의 원
> $sin\theta = b, cos\theta = a$
- 단위 원의 빗변 좌표 : 데카르트 좌표계에서 단위 원의 좌표는 $1 * (cos\theta,sin\theta)$표현할수 있다.
> x좌표는 $cos\theta$, y좌표는 $sin\theta$이므로 피타고라스 정리를 적용하면 $cos^2\theta + sin^2\theta = 1$이다
-  직각삼각형 변의 길이 : 길이가 1인 벡터와 평행하고, 길이는 r배 만큼 증가했으므로 스칼라 곱셉에 의해 $r \times (cos\theta, sin\theta)$라는 좌표를 가지게 됨.
> 직각삼각형 밑변의 길이 : $r \times cos\theta$, 직각삼각형 높이 : $r \times sin\theta$
- 삼각함수의 성질 : 단위원을 반시계방향으로 회전했을때 시작 위치가 $(1, 0) = 0° = (cos0°. sin0°)$
> 단위원이 90°까지 회전했을때 좌표값, $(0,1) = 90° = (cos90°, sin90°)$
- 진폭 : 변화 값의 범위
> 단위원은 진폭이 [-1, 1]
- 주기 : 360°마다 반복되는 각도
- 짝함수 : cos 함수와 같이 좌우 대칭 성질을 가진 함수
> $cos(-\theta) = cos(\theta)$
- 홀함수 : sin,tan 함수와 같이 원점 대칭 성질을 가진 함수
> $sin(-\theta) = -sin(\theta)$
- tan함수 : $\frac{sin\theta}{cos\theta}$  분모값이 0이 되는 90°에서는 값이 존재하지않는다.
- 각도법 : 각의 크기를 잴 때, 0~360까지의 수를 사용하는 방법
- 호도법 : 호의 길이를 기준으로 각을 측정하는 방법
>  $\pi$(rad) = 180°
## 게임 적용 
- 물체의 형상이 곡선으로 이루어져 있을때 삼각함수를 사용해 곡선으로 표현할수 있다.

## 코드예제
```c++
// 하트가 생성될 기준 좌표
Vector2 currentPosition;
// 하트가 생성될때 기본 크기
float currentScale = 10.f;

// 게임 로직을 담당하는 함수
void SoftRenderer::Update2D(float InDeltaSeconds)
{
	// 게임 로직에서 사용하는 모듈 내 주요 레퍼런스
	auto& g = Get2DGameEngine();
	const InputManager& input = g.GetInputManager();

	// 게임 로직의 로컬 변수
	// 생성된 하트 이동속도
	static float moveSpeed = 100.f;
	// 하트 최소 크기
	static float scaleMin = 5.f;
	// 하트 최대 크기
	static float scaleMax = 20.f;
	// 하트 크기를 조절할때 조절하는 속도
	static float scaleSpeed = 20.f;
	// 키보드 방향키로 생성된 하트의 기준 좌표를 이동시키는 코드.
	Vector2 inputVector = Vector2(input.GetAxis(InputAxis::XAxis), input.GetAxis(InputAxis::YAxis)).GetNormalize();
	// 하트가 이동할때 모니터 주사율차이에서 생기는 이동거리를 보정하는 코드
	Vector2 deltaPosition = inputVector * moveSpeed * InDeltaSeconds;
	// page up,page down 키를 눌러 하트의 z축 값을 변경하여 하트의 크기를 조절하는 코드(주사율 보정o)
	float deltaScale = input.GetAxis(InputAxis::ZAxis) * scaleSpeed * InDeltaSeconds;

	// 물체의 최종 상태 설정
	currentPosition += deltaPosition;
	// Math::Clamp() 함수 : 하트의 크기가 변할때 지정한 최솟값,최댓값을 넘지 못하도록 만드는 함수.
	currentScale = Math::Clamp(currentScale + deltaScale, scaleMin, scaleMax);
}

// 렌더링 로직을 담당하는 함수
void SoftRenderer::Render2D()
{
	// 렌더링 로직에서 사용하는 모듈 내 주요 레퍼런스
	auto& r = GetRenderer();
	const auto& g = Get2DGameEngine();

	// 배경에 격자 그리기
	DrawGizmo2D();

	// 렌더링 로직의 로컬 변수
	// 현재 각도
	float rad = 0.f;
	// 각을 서서히 증가시키는 증분값
	static float increment = 0.001f;
	// 하트의 형상을 나타낼 벡터들을 저장할 컨테이너 선언
	static std::vector<Vector2> hearts;

	// 하트를 구성하는 점 생성
	if (hearts.empty())
	{
		// Math::TwoPI : 2파이를 나타냄
		// 현재 각도에서 2파이까지 각도를 서서히 증가시키는 반복문을 생성
		for (rad = 0.f; rad < Math::TwoPI; rad += increment)
		{
			// 하트 방정식
			float sin = sinf(rad);
			float cos = cosf(rad);
			float cos2 = cosf(2 * rad);
			float cos3 = cosf(3 * rad);
			float cos4 = cosf(4 * rad);
			float x = 16.f * sin * sin * sin;
			float y = 13 * cos - 5 * cos2 - 2 * cos3 - cos4;
			// 하트의 형상을 나타낼 좌표들을 저장
			hearts.push_back(Vector2(x, y));
		}
	}
	// 컨테이너 hearts에 저장된 벡터들을 출력하여 화면에 표시
	for (auto const& v : hearts)
	{
		// 현재 벡터 값에 z축 스칼라 값을 곱해 하트의 크기를 정한뒤, 현재 위치 값을 더하여 하트의 위치를 결정. 하트의 색상은 파랑색
		r.DrawPoint(v * currentScale + currentPosition, LinearColor::Blue);
	}
	// 현재 하트의 크기와, 위치 값을 보여주는 코드
	r.PushStatisticText(std::string("Position : ") + currentPosition.ToString());
	r.PushStatisticText(std::string("Scale : ") + std::to_string(currentScale));
}
```
