# [2026.04.28] 벡터의 크기와 이동

## 핵심 개념
- 지정한 좌표를 기준으로 자신이 원하는 형상으로 벡터의 값을 제한을 걸고, 반복문을 수행하여 원하는 형상을 출력하도록 구현.


## 수학적 개념
- 벡터의 크기 : 원점으로 부터 최단거리
> 원점과 벡터를 연결해 직각삼각형을 그린 후, 피타고라스 정리를 사용하여 거리를 측정 
>>벡터 $V = (x, y)$의 크기: $||V|| = \sqrt{x^2 + y^2}$
- 단위 벡터 : 벡터의 크기가 1인 벡터
> 크기를 측정하는 기준.
- 정규화 : 임의의 벡터를 단위 벡터로 다듬는 작업
> 벡터의 크기로 나눠 단위 벡터를 만드므로, 벡터의 크기가 0보다 커야함
- 선형 연산 : 벡터의 합과 스칼라 곱셉처럼 일정하게 값이 증가되는 연산
- 선형 결합 : 선형 연산을 사용해 새로운 벡터를 생성하는 수식.
- 영벡터 : 벡터의 구성 원소가 0으로 구성된 벡터
- 선형 종속 관계 : 두 벡터의 곱하는 스칼라의 값이 0이 아니여도 영벡터가 나올수있는 관계
> ex) 2 * (1, 1) + (-1) * (2, 2) = (0, 0)

- 선형 독립 관계 : 두 벡터의 곱하는 스칼라의 값이 0이여야 영벡터가 나올수 있는 관계
> ex) 0 * (1, 2) + 0 * (2, 1) = (0, 0)
>> **선형 독립 관계인 벡터를 선형 결합 하면 벡터 공간에 속한 모든 벡터를 생성할수 있음**
- 기저 : 벡터 공간 내 모든 벡터를 생성할수 있는 선형 독립 관계를  가지는 벡터의 집합
- 기저 벡터 : 기저에 속하는 원소
- 표준 기저 : 단위 벡터로 구성된 집합
- 표준 기저 벡터 : 표준 기저의 원소
> ex) a = (1, 0), b = (0, 1)
- 차원 : 기저 벡터의 개수가 그 공간의 차원을 나타냄.
> ex) 2D 게임을 구현하려면 최소 2개의 선형 독립 관계를 가진 기저 벡터가 필요

## 게임 적용 
- 게임 속 물체의 형상이 지정한 좌표를 기준으로 생성
 ## 코드예제
```c++
/ 원의 중심 좌표
Vector2 currentPosition(100.f, 100.f);
// 렌더링 로직을 담당하는 함수
void SoftRenderer::Render2D()
{
	// 렌더링 로직에서 사용하는 모듈 내 주요 레퍼런스
	auto& r = GetRenderer();
	const auto& g = Get2DGameEngine();

	// 배경에 격자 그리기
	DrawGizmo2D();

	// 원의 반지름 50으로 지정.
	static float radius = 50.f;
	// 원을 구성하는 점을 보관하는 자료구조가 벡터형인 circles 선언.
	static std::vector<Vector2> circles;
	// 컨테이너 circles 요소가 비어있는 경우에만 원을 구성할수 있도록 조건문 설정.
	if (circles.empty())
	{
		// 지정한 원의 반지름 값을 넘지않는 x,y벡터값만 circles에 저장
		for (float x = -radius; x <= radius; ++x)
		{
			for (float y = -radius; y <= radius; ++y)
			{
				// (x,y) 값을 결합하여 벡터값 선언
				Vector2 pointToTest = Vector2(x, y);
				// 제곱근을 씌우지 않은 x^2 + y^2만 계산
				// 컴퓨터에게 제곱근 연산은 매우 무거운 작업. 거리의 비교만 필요하다면 제곱값끼리 비교하는것을 추천
				float squaredLength = pointToTest.SizeSquared();
				// 원의 넓이를 넘어서지 않는지 검사.
				if (squaredLength <= radius * radius)
				{
					// 위 조건들을 만족한 벡터값만 저장
					circles.push_back(Vector2(x, y));
				}
			}
		}
	}
	// circles에 저장된 모든 벡터 값을 출력하는 코드
	for (auto const& v : circles)
	{
        // v + currentPosition은 '벡터의 덧셈을 통한 평행이동'을 의미함
		r.DrawPoint(v + currentPosition, LinearColor::Red);
	}
	r.PushStatisticText("Coordinate : " + currentPosition.ToString());
}
```
