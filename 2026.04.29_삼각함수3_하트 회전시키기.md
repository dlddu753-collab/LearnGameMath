#[2026.04.29] 하트 회전시키기

## 핵심 개념
- 삼각함수를 이용해 사용자 입력에따라 지정한 각의 크기만큼 하트를 구성하는 모든 벡터를 회전시킨다.

- 표준 수학 좌표계: $\theta$가 커지면 반시계 방향으로 회전.
> 수학 좌표계에서 $(0, 1)$은 위쪽입니다. (3시 방향에서 12시 방향으로 갔으니 반시계)
- 화면 좌표계 (Y-Down): $\theta$가 커지면 시계 방향으로 회전.
> 화면 좌표계에서 $(0, 1)$은 아래쪽입니다. (3시 방향에서 6시 방향으로 갔으니 시계)  
> (코드작성후 콘솔화면에 출력하면 위에서 아래로 내려감 = 아래로 내려갈수록 커짐)

## 수학적 개념
- 표준가저벡터의 회전 : 각 $\theta$만큼 회전한 두 개의 표준기저벡터를 선형 결합하여 회전한 벡터의 좌표를 얻을수 있다.

>  $e_1 = (cos\theta,sin\theta)$ <- $(1, 0)$ 에서 $\theta$만큼 회전한 좌표
>>  $\theta$ = 90°가 되면 $(0, 1)$ 으로 좌표가 이동

>  $e_2 = (-sin\theta,cos\theta)$ <- $(0, 1)$ 에서 $\theta$만큼 회전한 좌표
>> $\theta$ = 90°가 되면 $(-1, 0)$ 으로 좌표가 이동

> 좌표 $(1,1)$ 이 각 $\theta$ 만큼 회전한 벡터의 좌표 $\vec{v}$
>> $\vec{v} = (cos\theta - sin\theta, sin\theta + cos\theta)$ 

- 임의의 벡터 회전 : $\vec{v} = (x,y)$ 에 대해 각 $\theta$ 만큼 회전한 벡터의 수식
>1. $\vec{v} = (x,y) = x\cdot e_1 + y \cdot e_2 = x \cdot (1, 0) + y \cdot (0, 1)$
>2. $\vec{v'} = (x',y') = x \cdot (cos\theta, sin\theta) + y \cdot (-sin\theta,cos\theta)$  
$= (xcos\theta - ysin\theta, xsin\theta + ycos\theta)$  
>$$x' = xcos\theta - ysin\theta$$
>$$y' = xsin\theta + ycos\theta$$


## 게임 적용 
- 게임속 캐릭터나 오브젝트를 회전하고자 할때 삼각함수를 사용해 회전을 구현

## 코드예제
```c++
Vector2 currentPosition;
float currentScale = 10.f;
// 하트의 현재 회전해야할 각도
float currentDegree = 0.f;

void SoftRenderer::Update2D(float InDeltaSeconds)
{
	auto& g = Get2DGameEngine();
	const InputManager& input = g.GetInputManager();

	static float moveSpeed = 100.f;
	static float scaleMin = 5.f;
	static float scaleMax = 20.f;
	static float scaleSpeed = 20.f;
	// 하트의 회전 속도
	static float rotateSpeed = 180.f;

	Vector2 inputVector = Vector2(input.GetAxis(InputAxis::XAxis), input.GetAxis(InputAxis::YAxis)).GetNormalize();
	Vector2 deltaPosition = inputVector * moveSpeed * InDeltaSeconds;
	float deltaScale = input.GetAxis(InputAxis::ZAxis) * scaleSpeed * InDeltaSeconds;
	// home,end 키를 입력해 변화된 각도를 계산하고 변수에 저장
	float deltaDegree = input.GetAxis(InputAxis::WAxis) * rotateSpeed * InDeltaSeconds;

	// 물체의 최종 상태 설정
	currentPosition += deltaPosition;
	currentScale = Math::Clamp(currentScale + deltaScale, scaleMin, scaleMax);
	currentDegree += deltaDegree;
}
	// 각도에 해당하는 사인과 코사인 값 얻기
	float sin = 0.f;
	float cos = 0.f;
	Math::GetSinCos(sin, cos, currentDegree);
	// 각 값을 초기화한 후 색상을 증가시키면서 점에 대응
	rad = 0.f;
	for (auto const& v : hearts)
	{
		// 1.점에 크기를 적용한다.
		Vector2 scaledV = v * currentScale;
		// 2. 크기가 변한 점을 회전시킨다.
		Vector2 rotatedV = Vector2(scaledV.X * cos - scaledV.Y * sin, scaledV.X * sin + scaledV.Y * cos);
		// 3. 회전시킨 점을 이동한다.
		Vector2 translatedV = rotatedV + currentPosition;

		hsv.H = rad / Math::TwoPI;
		r.DrawPoint(translatedV , hsv.ToLinearColor());
		rad += increment;
	}

	// 현재 위치,스케일,각도를 화면에 출력
	r.PushStatisticText(std::string("Position : ") + currentPosition.ToString());
	r.PushStatisticText(std::string("Scale : ") + std::to_string(currentScale));
	r.PushStatisticText(std::string("Degree : ") + std::to_string(currentDegree));
}
```
