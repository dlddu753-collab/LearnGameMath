# [2026.04.29] 두근대는 하트 애니메이션 만들기

## 핵심 개념
- 삼각함수를 시간에 따라 각을 서서히 증가시킨다면 시간에 따라 부드럽게 반복되는 애니메이션을 구현할수있다.

## 수학적 개념
- 선형 보간 : 두 지점 사이의 값을 직선 거리 비율에 따라 계산하는 방식
> $$P = A + (B - A) \times t$$
>(단, $t$는 보통 $0$에서 $1$ 사이의 값)
> - $t = 0$이면 결과는 $A$입니다.
> - $t = 1$이면 결과는 $B$입니다.
> - $t = 0.5$이면 결과는 $A$와 $B$의 정확한 중간값입니다.
- 재매핑 : 특정 범위($-1 \sim 1$)의 값을 다른 범위($0 \sim 1$)로 변환하는 과정.
## 게임 적용 
- 게임에 생성된 캐릭터나,오브젝트 움직임을 삼각함수를 사용하여 부드러운 움직임 구현.
 ## 코드예제
```c++
void SoftRenderer::Update2D(float InDeltaSeconds)
{
	// 게임 로직에서 사용하는 모듈 내 주요 레퍼런스
	auto& g = Get2DGameEngine();
	const InputManager& input = g.GetInputManager();

	// 물체의 이동속도
	static float moveSpeed = 100.f;
    // 물체의 최소 크기
	static float scaleMin = 5.f;
    // 물체의 최대 크기
	static float scaleMax = 20.f;
    // 애니메이션 왕복 시간
	static float duration = 1.5f;
    // 현재 경과 시간
	static float elapsedTime = 0.f;
    // 움직임 구현
	Vector2 inputVector = Vector2(input.GetAxis(InputAxis::XAxis), input.GetAxis(InputAxis::YAxis)).GetNormalize();
    // 모니터 주사율 차이 보정
	Vector2 deltaPosition = inputVector * moveSpeed * InDeltaSeconds;
	// 프레임당 경과 시간을 더해 현재 경과 시간을 갱신
	elapsedTime += InDeltaSeconds;
    // Math::FMod() : 나머지 값을 계산하는 함수.
    // 시간을 0 ~ 1.5초 사이로 제한
	elapsedTime = Math::FMod(elapsedTime, duration);
    // elapsedTime / duration: 현재 시간이 전체 주기에서 어느 정도 지났는지 비율(0.0 ~ 1.0)을 구합니다.
    // * Math::TwoPI: 그 비율을 원 한 바퀴($2\pi$, 약 $6.28$)에 매핑합니다.
	float currentRad = (elapsedTime / duration) * Math::TwoPI;
    /*
    sinf(currentRad): 결과값은 -1.0 ~ 1.0입니다. (이대로는 Lerp에 못 씁니다.)
    + 1.f: 값의 범위를 0.0 ~ 2.0으로 들어 올립니다. 
    * 0.5f: 값의 범위를 0.0 ~ 1.0으로 줄입니다.
    */
	float alpha = (sinf(currentRad) + 1) * 0.5f;
	// 물체의 최종 상태 설정
	currentPosition += deltaPosition;
    // alpha가 1.0일 때: currentScale은 scaleMax가 됩니다.
    // alpha가 0.0일 때: currentScale은 scaleMin이 됩니다.
	currentScale = Math::Lerp(scaleMin, scaleMax, alpha);
}
```
## 정리 및 요약
- Lerp의 $t$값(비율) 자리에 삼각함수를 넣으면, 단순한 선형 움직임을 생동감 넘치는 '왕복 애니메이션'으로 바꿀 수 있다.
