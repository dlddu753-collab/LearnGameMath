# [2026.04.30] 소용돌이 효과 구현

## 핵심 개념
- 극좌표계를 활용해 설정한 각의 값에 따라 벡터를 회전시켜 소용돌이 효과를 구현
> 중심에서 멀어지는 벡터들한테 더 많은 회전량을 부여해 소용돌이 효과 부여


## 수학적 개념
- 전사함수 : 공역의 모든 요소가 정의역에 대응 되는 함수
- 단사함수 : 정의역과 공역의 요소가 일대일로 대응되는 함수
- 전단사함수 : 정의역과 공역의 *모든* 요소가 빠짐없이 일대일 대응되는 함수
- 항등함수 : 정의역과 공역이 동일한 값으로 대응되는 함수
- 역함수 : 합성함수의 결과가 항등함수가 되는 함수
> 역함수가 되려면 함수의 형태가 **전단사함수** 의 형태가 되어야 한다
- 삼각함수의 역함수 : 삼각함수의 정의역과 공역을 제한 시켜 전단사함수형태로 만든 함수
> $\sin^{-1}(x)$: 정의역 $[-1, 1]$, 공역 $[-90^\circ, 90^\circ]$ (혹은 $[-\frac{\pi}{2}, \frac{\pi}{2}]$)   
> $\cos^{-1}(x)$: 정의역 $[-1, 1]$, 공역 $[0^\circ, 180^\circ]$ (혹은 $[0, \pi]$)  
> $\tan^{-1}(x)$: 정의역 실수 전체($\mathbb{R}$), 공역 $(-90^\circ, 90^\circ)$ (혹은 $(-\frac{\pi}{2}, \frac{\pi}{2})$)
- 극좌표계 : 회전 동작을 기반으로 설계된 좌표계
> 극좌표계는 원점으로부터의 거리 r,각 $\theta$ 두 요소로 구성. 극좌표계의 표시 = $(r,\theta)$
- 데카르트 좌표계 -> 극좌표계 : 데카르트 좌표계로 표현된 벡터$(x,y)$는 벡터의 크기와 $tan^{-1}$함수를 사용하여 극좌표계 $(r,\theta)$로 변환할 수 있다.
> $$\tan\theta = \frac{y}{x}$$
> $$\tan^{-1}(\tan\theta) = \tan^{-1}\left(\frac{y}{x}\right)$$
> $$\theta = \tan^{-1}\left(\frac{y}{x}\right)$$
- 극좌표계 -> 데카르트 좌표계 : 극좌표계 좌표 $(r,\theta)$ 를 데카르트 좌표계 $(x,y)$로 변환하는 식은 삼각함수를 사용해 구할수 있다.
> $$x = r\cdot cos\theta$$
> $$y = r\cdot sin\theta$$


## 게임 적용 
- 게임속 소용돌이 이펙트를 가진 스킬이나 구조를 극좌표계를 활용해 구현할수 있다
- `atan2(y,x)`는 $x, y$의 부호를 모두 따져 $360^\circ$($-\pi \sim \pi$) 전 범위를 반환
 ## 코드예제
```C++
// 렌더링 로직을 담당하는 함수
void SoftRenderer::Render2D()
{
	// 렌더링 로직에서 사용하는 모듈 내 주요 레퍼런스
	auto& r = GetRenderer();
	const auto& g = Get2DGameEngine();

	// 배경에 격자 그리기
	DrawGizmo2D();

	// 렌더링 로직의 로컬 변수
	// 정사각형의 절반 크기를 변수에 저장
	static float halfSize = 100.f;
	// 정사각형을 구성하는 벡터를 담을 컨테이너 변수 선언
	static std::vector<Vector2> squares;

	// 사각형을 구성하는 점을 생성
	if (squares.empty())
	{
        // 사각형의 중심이 좌표계 원점에 생성되도록 음수 -> 양수 벡터값 설정
		for (float x = -halfSize; x <= halfSize; x += 0.25f)
		{
			for (float y = -halfSize; y <= halfSize; y += 0.25f)
			{
				squares.push_back(Vector2(x, y));
			}
		}
	}
	
	float sin = 0.f;
	float cos = 0.f;
    // 현재 각도에 해당하는 sin과 cos 값 얻기
	Math::GetSinCos(sin, cos, currentDegree);

	//현재 화면의 크기로부터 길이를 비교할 기준양 정하기
	static float maxLength = Vector2(_ScreenSize.X, _ScreenSize.Y).Size() * 0.5f;

	// 사각형 그리기
	HSVColor hsv(0.f, 1.f, 0.85f);
	for (auto const& v : squares)
	{
		// 극좌표계로 변경. v(원점으로 부터 거리, 각)
		Vector2 polarV = v.ToPolarCoordinate();

		// 극좌표계의 각 정보로부터 색상을 결정
		if (polarV.Y < 0.f)
		{
			polarV.Y += Math::TwoPI;
		}
		hsv.H = polarV.Y / Math::TwoPI;

		// 극좌표계의 크기 정보로부터 회전량을 결정
		float ratio = polarV.X / maxLength;
		float weight = Math::Lerp(1.f, 5.f, ratio);

		// 극좌표계를 사용해 회전을 부여
		polarV.Y += Math::Deg2Rad(currentDegree) * weight;

		// 최종 값을 데카르트 좌표계로 변환
		Vector2 cartesianV = polarV.ToCartesianCoordinate();
		r.DrawPoint(cartesianV, hsv.ToLinearColor());
	}

	// 현재 각도를 화면에 출력
	r.PushStatisticText(std::string("Degree : ") + std::to_string(currentDegree));
}
```
